* [Задача 3](https://github.com/nelsonnetru/python/tree/main/homework_02/003.py): Петя и Катя – брат и сестра. Петя – студент, а Катя – школьница. Петя помогает Кате по математике. | X * Y = P

```
Y**2 - S*Y + P = 0
```
